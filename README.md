# LyScript MCP

<img width="10%" height="10%" alt="logo" src="https://github.com/user-attachments/assets/5b1af7e5-4216-4a6b-9c87-1bb0b876aa71" />

LyScript 是一款专为 x64dbg 调试器打造的自动化测试插件，以 Python 为基础构建测试脚本，既能依托 Python 丰富的第三方库结合调试器特性提升开发效率，又具备无额外第三方依赖、开箱即用的便捷性，同时支持调用 x64dbg 原生脚本及自定义组合函数，可有效加速漏洞利用程序开发、辅助漏洞挖掘并助力恶意软件分析，为相关技术研发与安全分析工作提供高效支持。
